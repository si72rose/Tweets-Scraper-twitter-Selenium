# Tweets Scraper from twitter in Python Using Selenium
Recently twitter has stopped giving thier Apis to everyone, So i write simple Python script using selenium to scrap  all the tweets from a user and store into a csv file 
# Extracting Tweets from the #user in twitter

## Usage
Twitter allow to see only recent 300 tweets from any user.In order to access the Url of the user with all the past post follow question and answer in here [Stackexchange](https://webapps.stackexchange.com/questions/112171/how-to-see-other-users-old-tweets) and search in tweeter search bar copy the URL of the search page and paste into tweet_scraper.py file
